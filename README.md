# Hiren-MailBox [![Requirements Status](https://requires.io/github/pyprism/Hiren-MailBox/requirements.svg?branch=master)](https://requires.io/github/pyprism/Hiren-MailBox/requirements/?branch=master)
A simple webmail client based on Mailgun's api.

After login to application users would be able to do normal task like managing, sending and receiving mail.

### Requirements
- Python 3.6
- PostgreSQL
- Web Server (eg: Apache, Nginx)
